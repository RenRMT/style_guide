# Dot density 

represent the distribution of discrete phenomena with point symbols (each denoting the same quantity). Means of visualisation: colour, shape, position

# Proportional symbols

show differences in absolute quantities through size differences. means of visualisation: colour, shape, size, position.  Note: use transparency if symbols overlap

# Diagram

use diagrams to display additional information for specific points or polygons

# visual variables

-	position
-	size
-	shape (2-3 features)
-	value / colour (hue, saturation, luminosity) (4-6 variables)
-	orientation (of pattern) (7-8 variables)
-	texture (variations in grain/density) (9-12 variables)
